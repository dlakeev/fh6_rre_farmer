Данный скрипт может быть использован в качестве прокачки уровня в Forza Horizon 6's Raku Raku Express. К сожалению, для многих этот режим оказался довольно скучным - миссия повторяются, а для достижения нужно получить много опыта. С этим может помочь данный скрипт:

1) Скрипт автоматически продолжает смену.
2) Скрипт автоматически включает автопилот, если он выключен.

# Обратите внимание
!!!Включен автопилот или нет - определяется по области на интерфейсе игры (CROP_REGION). Для разрешения 2560x1440 с 125% масштабированием это область (115, 1325, 270, 1390). Вы можете изменить это значение для себя. Надо, чтобы было видно template.!!!

!!!Автопилота достаточно для того, чтобы получать 1-2 звезды за задание. Да, это займет больше времени, так как ты не получаешь бонусы за 3 звезды, но зато здесь все происходит без твоего участия.!!!

!!!В миссиях с дрифтом или набором очков автопилот не работает, а по их окончанию автопилот отключается. Скрипт дождется окончания миссии и вновь запустит автопилот.!!!
